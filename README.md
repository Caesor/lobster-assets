# 龙虾营素材仓库

本仓库用于托管对外公开素材（PDF / 封面图 / 下载资源），通过 jsDelivr CDN 加速。

## 当前素材

- `pdf/` — PDF 下载资源
- `images/` — 图片素材（封面图、宣传图等）

## 访问方式

- jsDelivr CDN: `https://cdn.jsdelivr.net/gh/Caesor/lobster-assets@main/<path>`
- GitHub Releases: `https://github.com/Caesor/lobster-assets/releases/download/<tag>/<file>`
